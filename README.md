# Ejecutando el proyecto

1. clonar el repositorio
```
git clone https://github.com/velfin13/pokedex.git
```
2. nos movemos a la carpeta del proyecto
 ```
cd pokedex
```

3. instalar las dependencias
```
npm install
```

4. levantar el proyecto en modo desarrollo
```
npm run dev:start
```
